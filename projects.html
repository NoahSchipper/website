Projects

Schipper Statlines is a full-stack baseball statistics database that allows users to compare player and team stats.

This tool leverages PostgreSQL and SQLAlchemy to efficiently retrieve and process comprehensive baseball statistics, delivering instant player and team comparisons through an intuitive interface.

Due to hosting constraints, some player comparison features had to be cut, but a video demonstration of player comparisons with more detailed features can be viewed by clicking the video demo button.

Fragrance Finder is an AI powered tool to help users explore different fragrances. It offers three different modes: search by similarity, search by accords, and random.

The search by similarity mode uses a hugging-face AI model to assign cosine similarities to each fragrance, suggesting similar options to what the user entered.

The search by accord takes in a user entered scent (woody, citrus, fruity, etc.) and shows fragrances featuring that scent as their top accord.

Schipper Translate is a powerful language translation extension offering in-extension, in-text, and tooltip translation options.

Schipper Translate uses the Deep Translate API to support almost 100 languages and features a minimal, non-invasive design.

Perfect for browsing foreign websites, reading international news, or learning new languages with seamless translation features built right into your browser.
