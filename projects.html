Projects

Fragrance Finder is an AI powered tool to help users explore different fragrances. It offers three different modes: search by similarity, search by accords, and random.

The search by similarity mode uses a hugging-face AI model to assign cosine similarities to each one of the avaliable fragrances. These similarities are then used to suggest fragrances similar to one that the user entered.

The search by accord takes in a user entered scent (woody, citrus, fruity, etc.) and shows fragrances featuring that scent as their top accord.

Schipper Statlines is a baseball statitics database that allows users to compare player and team stats.

Due to hosting constraints, some player comparison features had to be cut, but a video demonstration of player comparisons with more detailed features can be viewed by clicking the bottom button to the right.

Schipper Translate is a powerful language translation tool offering in-extension, in-text, and tooltip translation options.
